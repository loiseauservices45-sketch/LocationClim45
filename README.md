# LocationClim45 — Site Next.js 14

## Stack
- Next.js 14 App Router
- TypeScript
- Tailwind CSS
- Vercel (déploiement recommandé)

## Pages
- `/` — Accueil
- `/particuliers` — Page particuliers
- `/professionnels` — Page professionnels
- `/climatiseurs` — Catalogue
- `/zone-livraison` — Zone de livraison
- `/contact` — Contact & devis

## Lancer en local
```bash
npm install
npm run dev
```

## Déployer sur Vercel
1. Pusher ce dossier sur GitHub
2. Importer le repo sur vercel.com
3. Laisser les paramètres par défaut (Next.js détecté automatiquement)
4. Cliquer Deploy
